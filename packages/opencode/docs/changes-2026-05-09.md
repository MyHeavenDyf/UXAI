## 一、功能变更概述

本次修改包含三大核心功能：

1. **build → octo_ai 重命名** — 将默认 agent 从 `build` 重命名为 `octo_ai`
2. **Session Category 分类系统** — 为 session 添加分类功能，按 agent 类型自动归类
3. **新增 4 个专业化 Agent** — octo_insight、octo_make、octo_design、octo_canva

---

## 四、数据库迁移

**migration.sql**:

```sql
CREATE TABLE `session_category` (
  `session_id` text PRIMARY KEY REFERENCES `session`(`id`) ON DELETE CASCADE,
  `category` text NOT NULL,
  `time_created` integer,
  `time_updated` integer
);
CREATE INDEX `session_category_category_idx` ON `session_category` (`category`);
```
