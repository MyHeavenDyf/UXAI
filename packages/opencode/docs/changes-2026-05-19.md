## 二、提交记录

### Commit: `0998e5990` — fix: 防止 chat 页面 session 创建竞态导致重复 session

**问题**: 从 cowork/insight 模式切换到 chat 模式时，创建 session 对话会卡住，多次按 Enter 后突然出现多个重复 session。

**根因**: `submit.ts` 的 `handleSubmit` 中新 session 创建期间无互斥锁。`working()` 在新 session 时为 false（无 session ID），导致多次 Enter 并发创建多个 session。

**修改内容**:

| 文件 | 改动 |
|------|------|
| `packages/app/octoapp/components/prompt-input/submit.ts:215` | 添加 `let creatingSession = false` 互斥锁 |
| `submit.ts:293` | 入口检查 `if (creatingSession) return` |
| `submit.ts:368-390` | session 创建段用 `try/finally` 包裹设置/清除标志 |
